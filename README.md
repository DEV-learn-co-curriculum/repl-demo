### Ruby Repl

Write a method that reverses an string, and call it, passing '12345' as an argument.

~~~ruby

def reverse(str)
end

~~~solution

def reverse(string)
  string.reverse
end

reverse("12345")

~~~validation

assert.strictEqual(response,"54321");

~~~
